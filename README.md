# betmax

A Next.js project — Betmax

## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | Next.js |
| Language | TypeScript |
| Styling | Tailwind CSS |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Ross-Ward/betmax.git
cd betmax
```

```bash
npm install
npm run dev
```

## Deployment

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/Ross-Ward/betmax)
